	ОБЛАСТЬ МЕНЮ

SAVE		сохранить схему категорий и их товарный состав
LOAD		загрузить схему категорий и их товарный состав
CLEAR		очистить схему категорий и их товарный состав
ZOOM IN		увеличить масштаб
ZOOM OUT	уменьшить масштаб
SWITCH		включить/выключить панель товаров
REPORT		загрузить/отключить режим отчета
BARCODE		сортировать товары по возрастанию кода
DESCRIPTION	сортировать товары в алфавитном порядке
DEFAULT		сортировка товаров по умолчанию
SNAPSHOT	сохранить скриншот для печати

NO CATEGORY	содержит не распределенные по категориям товары

	ОБЛАСТЬ КАТЕГОРИЙ

			RightClick		добавить категорию
RightAlt	+	LeftClick		изменить наименование категории
LeftControl	+	LeftClick		связать родителя с потомком
			LeftClick удерживая	перемещение категории
RightShift					подсветка всех потомков
RightShift	+	LeftClick удерживая	перемещение ветви
LeftControl	+	LeftClick		разорвать связь с родителем
LeftAlt		+	LeftClick		разорвать связи с потомками
LeftShift	+	LeftClick		удалить категорию
Up/Down/Right/Left + opt. LeftControl		скроллинг

---------------------------------------------------------------

	ОБЛАСТЬ ТОВАРОВ

			LeftClick удерживая	перемещение товара в категорию
LeftControl	+	LeftClick		выделение товаров
LeftControl	+	LeftClick удерживая	перемещение выделенных товаров
